# 微光工坊 - 深海夜光装饰网站

## 项目概述

一个面向北美市场的深海主题夜光装饰产品展示网站，采用沉浸式深海视觉设计，支持后台内容管理。

## 技术栈

- **前端**: HTML5 + CSS3 + JavaScript (原生)
- **数据存储**: LocalStorage (浏览器本地存储)
- **后台管理**: 纯前端实现，无需后端服务器

## 使用说明

### 首次使用

1. 打开 `index.html` 查看前台页面
2. 打开 `admin/index.html` 进入后台管理
3. 后台会自动初始化默认产品数据

### 设置视频背景

1. 进入后台 → 视频管理
2. 上传视频文件 (MP4格式，建议10-30秒)
3. 点击"设为首页背景"
4. 返回前台查看效果

### 添加产品图片

1. 进入后台 → 图片管理
2. 上传产品图片
3. 进入产品管理 → 编辑产品
4. 点击"选择图片"
5. 保存后前台自动更新

### 添加新产品

1. 进入后台 → 产品管理
2. 点击"添加产品"
3. 填写产品信息
4. 选择产品图片 (可选)
5. 保存后前台自动显示

## 数据存储说明

所有数据存储在浏览器 LocalStorage 中：

| Key | 说明 |
|-----|------|
| `images` | 上传的图片列表 |
| `videos` | 上传的视频列表 |
| `products` | 产品数据 |
| `settings` | 网站设置 |
| `cart` | 购物车数据 |

**注意**: 
- 数据仅在当前浏览器有效
- 清除浏览器数据会丢失所有内容
- 建议使用相同浏览器管理前后台

## 设计特点

### 色彩方案
- 主色: #0a1628 (深海蓝)
- 强调色: #00d9ff (青蓝发光)
- 辅助色: #c77dff (紫罗兰)
- 文字: #e0e1dd (米白)

### 动画效果
- 气泡上升: 15s 无限循环
- 光球漂浮: 20s 缓动
- 卡片悬停: 发光 + 上浮
- 滚动渐显: IntersectionObserver
